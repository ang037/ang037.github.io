ROADIES Ongoing
Jan 2023 – Present  ·  Evolutionary Genomics / HPC
Reference free, Orthology free, Alignment free, Discordance aware Estimation of Species tree — a fully automated, end-to-end HPC pipeline for inferring species trees directly from raw genome assemblies. Uses GPU/CPU co-processing and parallelization primitives to achieve massive scalability.
  • Achieved 176× speedup in tree generation for large-scale genomics datasets
  • Published in PNAS 2025, featured on the journal cover
  • Presented at ISMB 2024, Montreal, Canada
PNAS 2025 Website GitHub
DP-HLS HPCA 2026
Nov 2022 – Jun 2024  ·  FPGA Acceleration / Bioinformatics
An HLS-based open-source framework that streamlines the creation of FPGA accelerators for dynamic programming algorithms used in bioinformatics — covering sequence alignment, homology search, basecalling, and more. Features a parameterizable systolic array architecture allowing algorithm customization without hardware redesign.
  • Achieved 32× speedup over CPU/GPU baselines and 20× faster development time than hand-coded HDL
  • Implemented 15 diverse DP kernels; compatible with AWS EC2 F1 FPGA instances
arXiv Website GitHub
ML Hardware Acceleration Ongoing

Designing hardware-aware weight compression techniques to significantly optimize and accelerate Deep Learning inference on specialized hardware. Focused on maximizing computational efficiency and reducing memory footprint for deploying large-scale neural networks at the edge.
